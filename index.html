О проекте

ARDA Student Manager — это мощный инструмент на C++ и Qt, который решает проблему учёта студентов: от добавления до дипломов. Работает локально и может быть расширен для сетевого режима. Все данные сохраняются, можно выгрузить в JSON или распечатать.

Функционал

  • Менеджер студентов: добавление, удаление, поиск, сортировка
  • Конкурсы (аттестаты, ЕГЭ для 9-11 классов)
  • Учёт оценок, курсовых, дипломных работ
  • Сохранение/загрузка в JSON
  • Сериализация в реестр Windows
  • Экспорт в PDF, формат A4
  • Будущее: файловый менеджер, админ-панель

Краткое руководство по установке и сборке:

git clone https://github.com/Gerakl1123/ARDA_Stud_Manager
cd ARDA_Stud_Manager
mkdir build && cd build
cmake ..
make

Минимальные требования:

  • Qt 5.15+ или Qt 6
  • CMake
  • Компилятор с поддержкой C++17
