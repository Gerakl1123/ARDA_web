О приложении

ARDA Student Manager — это приложение на C++/Qt, которое помогает вести учёт студентов локально или в сети. Позволяет добавлять, удалять, сортировать и находить студентов. Также содержит модули для конкурсного учёта (аттестаты, ЕГЭ для 9–11 классов), оценок, проектной деятельности, зачетов, курсовых и дипломов.

Основные функции

  • CRUD‑операции: добавление, удаление, редактирование, поиск студентов.
  • Сортировка и фильтрация данных.
  • Меню конкурсы: аттестаты и ЕГЭ (9–11 классы).
  • Учёт оценок: оценки, проектная деятельность, зачеты, курсовые, диплом.
  • Сохранение в JSON + выгрузка.
  • Отображение данных в таблицах.
  • Сериализация в реестр Windows.
  • Печать в PDF, формат A4.
  • В разработке: файловый менеджер (создание директорий и т.п.) и админ‑панель.

Описание

ARDA Student Manager — кроссплатформенное приложение на C++17 и Qt (версии 5.15 / 6.x) с использованием CMake. Поддерживает Windows и Linux. Приложение работает полностью офлайн, хранит данные в локальных файлах.

Как собрать

git clone https://github.com/Gerakl1123/ARDA_Stud_Manager
cd ARDA_Stud_Manager
mkdir build && cd build
cmake ..
make
./ARDA_Stud_Manager

Требования

  • Qt и CMake в PATH.
  • Компилятор с поддержкой C++17.
